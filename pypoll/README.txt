The purpose of this code was to analyze a small rural town's voting data.  The code was used to analyzie an existing set of data provided.  The data set was composed as three columns which included "Voter ID", "County", and "Candidate".  This Python script analyzes the votes and provides the following:

1.  Total number of votes cast
2.  List of candidates that received votes
3.  The percentage of votes each candidate won
4.  The total number of votes each candidate one
5.  The winner of the election based on popular vote
